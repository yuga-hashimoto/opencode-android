OpenCode Android

Run AI coding agents directly on your Android device, or connect remotely to OpenCode on your PC. No cloud lock-in.

↓ Download APK View on GitHub

Features

📱

On-Device Runtime

Full Alpine Linux + OpenCode running on your phone via PRoot. No PC required — code anywhere.

🖥️

Remote PC Connection

Connect to OpenCode on your Mac/PC/Linux over LAN or Tailscale with zero-config mDNS discovery.

🔄

Seamless Handoff

Switch between local and remote execution mid-conversation. Start on the train, finish at your desk.

🎙️

Voice + Wake Word

Push-to-talk voice input with wake word detection. Use it as your Android digital assistant.

🔒

Security First

Credentials in Android Keystore. Dangerous operations require explicit approval. No auto-approve.

⚡

Real-Time Streaming

Live SSE responses with collapsible reasoning, tool calls, and command output in a structured timeline.

Get Started in Minutes

1

Download the APK

Grab the latest release from GitHub. Supports arm64 and x86_64 devices.

2

Choose your runtime

Set up the on-device runtime (2 min) or connect to your PC's OpenCode server via LAN/Tailscale.

3

Start coding

Chat with your AI agent, approve tool operations, switch models — all from your phone.
